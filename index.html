Album setup

One-time setup to connect your Google Drive. Takes about 15 minutes!

Step 1 — Google Drive folder ID

Create a folder in Google Drive, share it with your partner's Google account, then copy the ID from the URL (the long string after /folders/).

Step 2 — Google OAuth Client ID

Go to console.cloud.google.com → New Project → Enable Google Drive API → Create OAuth 2.0 credentials (Web Application). Add your GitHub Pages URL as an authorized redirect URI.

Step 3 — Share the link 🌸

Host this file on GitHub Pages, send your partner the URL. She signs in with Google and can view and upload photos to your shared folder.
